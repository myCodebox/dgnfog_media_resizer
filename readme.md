# DGNFOG Media Resizer

A small image processor script written in go.

## Usage

Start the script
```bash
go run .
```

Confirm the process
```bash
Press "y" to start resizing: y
```

Here is the end screen
```bash
----------------------------------------
Yeah, done! \o/
36 Images are stored in folder "./src/out/"
----------------------------------------
```

Stop it with `Ctrl + C`
```bash
----------------------------------------
Why do you press Ctrl + C?
----------------------------------------
```
